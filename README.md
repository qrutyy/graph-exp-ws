# Graph Algorithm Performance Study

Two-block study comparing graph algorithm implementations.

---

## Block 1 — Linear Algebra (SPLA / GraphBLAS + LAGraph)

Compares CPU and GPU performance of BFS, SSSP, PageRank, and Triangle Counting
using [SuiteSparse:GraphBLAS + LAGraph](https://github.com/GraphBLAS/LAGraph) and
[SPLA](https://github.com/SparseLinearAlgebra/spla).

**Notebooks:** `docs/bfs.ipynb`, `docs/sssp.ipynb`, `docs/tc.ipynb`, `docs/pg.ipynb`  
**Source runners:** `src/`  
**Results:** `results/block1/` (see individual notebooks)

---

## Block 2 — Distributed Graph Processing (Galois)

Strong-scaling study measuring BFS, PageRank, and Connected Components across
1 → 2 → 4 → 6 → 8 MPI processes using
[Galois distributed](https://github.com/IntelligentSoftwareSystems/Galois/tree/master/lonestar/analytics/distributed).

**Notebook:** `docs/block2_galois.ipynb`  
**Experiment doc:** `docs/block2_distributed.md`  
**Scripts:** `scripts/run_experiments.py`, `scripts/analyse.py`  
**Results:** `results/block2/csv/`, `results/block2/plots/`

Raw `.mtx` files are in `datasets/`. Converted Galois `.gr` files are generated by
the experiment script and excluded from git (see `.gitignore`).

### Reproducing Block 2

**1. Build Galois** (on a Linux cluster or locally):

```bash
git clone --depth=1 https://github.com/IntelligentSoftwareSystems/Galois
cd Galois && mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release -DGALOIS_ENABLE_DIST=ON
make -j$(nproc) bfs-pull-dist pagerank-pull-dist connected-components-pull-dist graph-convert
```

**2. Convert graphs** (run once, requires `graph-convert` in PATH or `GALOIS_BUILD` set):

```bash
export GALOIS_BUILD=/path/to/galois/build
python scripts/run_experiments.py --convert-only   # or run the notebook conversion cell
```

**3. Run experiments:**

```bash
export GALOIS_BUILD=/path/to/galois/build
export MPIRUN=/path/to/mpirun          # optional, defaults to Homebrew MPICH
python scripts/run_experiments.py
```

**4. Generate plots:**

```bash
python scripts/analyse.py
# or open docs/block2_galois.ipynb and run sections 4–6
```

Results are written to `results/block2/csv/` and `results/block2/plots/`.
